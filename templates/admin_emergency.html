🚨 Herramientas de Emergencia

📊 Dashboard 👥 Usuarios

⚠️ PRECAUCIÓN

Estas herramientas modifican datos directamente. Úsalas solo cuando sea absolutamente necesario.

💰 Restaurar Balance

Agrega o establece el balance de un usuario manualmente.

👥 Agregar Referido Manual

Crea una relación de referido manualmente entre dos usuarios.

🔓 Desbanear por IP

Desbanea todos los usuarios asociados a una dirección IP.

🔄 Recalcular Referidos

Recalcula el contador de referidos de un usuario basándose en la base de datos.

⏱️ Resetear Timer de Minería

Permite al usuario volver a minar inmediatamente.

✅ Validar Referido Pendiente

Valida manualmente un referido que no se procesó correctamente.
